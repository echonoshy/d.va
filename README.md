# 思路
构建一个可以每日更新语音版新闻的助手，然后自动推送到播客和小红书涨粉。

重点是：
1. 有用
2. 涨粉

粗略的实现思路：
1. 每日获取不同领域的专业新闻
2. 讲新闻分解成新闻稿
3. 新闻稿变成语音稿件 （ 带片头片尾曲，时长不超过4分钟）
4. 自动推送到小红书和播客
